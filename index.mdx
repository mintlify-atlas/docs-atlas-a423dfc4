  <Steps>
    <Step title="Acceda al sistema">
      Inicie sesión con sus credenciales de usuario. El sistema soporta tres roles: Administrador, Analista y Desarrollador.
      
      ```bash
      POST /usuarios/login
      Content-Type: application/json

      {
        "indicador": "MALAVEN",
        "password": "su_contraseña"
      }
      ```
    </Step>
    <Step title="Cree su primer permiso">
      Acceda al módulo de permisos correspondiente y complete el formulario con la información del solicitante y los detalles del permiso.
      
      <Note>
        El sistema valida automáticamente los documentos de pago y genera códigos únicos para cada permiso basados en el año y número secuencial.
      </Note>
    </Step>
    <Step title="Gestione el flujo de aprobación">
      Los permisos pasan por un flujo de revisión donde pueden ser aprobados o cancelados. El sistema notifica en tiempo real mediante WebSockets.
      
      ```javascript
      // El sistema emite eventos en tiempo real
      socket.on('bebidas:nuevo-permiso', (data) => {
        // Actualización automática de la interfaz
        console.log('Nuevo permiso:', data);
      });
      ```
    </Step>
    <Step title="Genere reportes">
      Acceda al módulo de reportes para exportar permisos en formato PDF con toda la información registrada y el estado actual.
    </Step>
  </Steps>
</div>
